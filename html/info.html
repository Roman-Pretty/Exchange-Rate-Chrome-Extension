How to use

Enter the amount you want to convert in the input field. Then select the currency you want to convert from and to. Click the convert button and the result will be displayed below.

How it works

The currency converter uses the Exchange Rates API to get the latest exchange rates. The API is updated every 24 hours. The API is free to use non-commerically.
